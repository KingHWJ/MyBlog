# 进程、线程与内存

## 先别急着背定义

这三个词最容易被背成八股，但真正重要的是知道它们分别解决什么问题。

## 一个实用理解

- 进程：资源分配的基本单位
- 线程：CPU 调度的基本单位
- 内存：程序状态与数据的承载空间

## 为什么后端开发要关心

因为线上问题经常会落到这些维度，例如 CPU 飙高、内存泄漏、线程阻塞。
