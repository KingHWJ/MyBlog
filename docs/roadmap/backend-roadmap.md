# 后端学习路线

## 第一阶段：语言与运行时

- Java 基础语法、集合、并发、JVM
- 学会把“会写代码”升级成“理解程序如何运行”

## 第二阶段：数据与存储

- 数据模型设计
- MySQL 索引、事务、锁与 SQL 优化
- Redis 等缓存组件的基础用法与设计场景

## 第三阶段：系统与网络

- TCP/IP 与 HTTP 协议
- Linux 常见进程、线程、内存和 IO 基础
- 常见线上排查思路

## 第四阶段：工程与体系化复盘

- 把碎片知识整理成专题文档
- 用更新文章记录问题、方案和复盘
- 逐步形成适合自己的知识库
