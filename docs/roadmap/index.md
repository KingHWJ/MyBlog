# 学习路线总览

这部分不追求一次列全，而是提供一个可持续迭代的学习框架。

## 推荐阅读顺序

1. 先完成后端基础主线，建立 Java、数据库、网络、操作系统的整体认知。
2. 在具备基础之后，再切入大数据生态，理解离线与实时处理链路。
3. 遇到新的场景时，用更新文章承接，再回填到专题目录。
